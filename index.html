Index

  1. Thoughts On Interaction of Color

    A project responding to Chapter 16 of Interaction of Color by Josef Albers, involving typesetting and replying to the text as a web page.

  2. Cult of the Ugly

    A collaborative project responding to Cult of the Ugly by Steven Heller as a web page, involving cooperative sketching, pair programming and creating a print layout.

  3. Persistence of Memory

    A project that curates three readings as a digital binding on a website, involving sketching in code and developing a design system.

  4. Gossip

    A collaborative project using the Are.na API to build a gossip-themed website that transforms a curated archive into an interactive interface.
